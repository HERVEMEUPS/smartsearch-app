ÉTAPE 2️⃣ : Se Connecter en tant qu'Admin
────────────────────────────────────────
1. Ouvrir le navigateur
2. Aller sur: http://localhost:3000/frontend/login.html
3. Entrer:
   Username: admin
   Password: admin
4. Cliquer sur "Se connecter"


ÉTAPE 3️⃣ : Ouvrir le Dashboard
────────────────────────────────
Aller sur: http://localhost:3000/frontend/dashboard.html


ÉTAPE 4️⃣ : Voir les Correspondances

───────────────────────────────────────────────────────────────────────────
❓ PROBLÈMES COURANTS
───────────────────────────────────────────────────────────────────────────

Problème : "Identifiants incorrects"
Solution : Utilisez admin / admin (en minuscules)

Problème : "Erreur réseau" ou "Failed to fetch"
Solution : Vérifiez que le backend tourne sur le port 3000

Problème : "Aucune correspondance détectée"
Solution : Actualisez la page (F5) ou cliquez sur 🔄 Actualiser

Problème : Anciennes données affichées
Solution : Videz le cache (Ctrl + Shift + R)
